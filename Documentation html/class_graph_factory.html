Calcule le prix du trajet entre deux stations en fonction de la ligne utilisée.

Parameters
debutLa station de départ (ne doit pas être null).
arriveLa station d'arrivée (ne doit pas être null).
lignesLa liste des lignes disponibles (ne doit pas être null ou vide).
Returns
Le prix du trajet en euros. Retourne 0 si aucune ligne ne relie les deux stations.
Exceptions
